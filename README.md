# Game Mechanic Compliance Pre-screener

## Overview
The **Game Mechanic Compliance Pre-screener** is an AI Skill designed for Game Producers and Product Managers. It acts as an expert Mobile Gaming Compliance Specialist, evaluating proposed monetization mechanics, gacha systems, and live-ops events against global platform policies and regulations *before* a single line of code is written. 

This proactive screening saves studios thousands of dollars in development time and prevents devastating app store rejections or legal fines.

## The Problem It Solves
Modern mobile game monetization is a minefield of regulations. Designing a mechanic that inadvertently violates Apple App Store Guidelines, Google Play Policies, or EU/UK consumer protection laws can lead to:
- App store rejections during critical launch windows.
- Wasted development cycles building non-compliant features.
- Fines or lawsuits related to deceptive patterns or minor protection laws (COPPA).

This skill provides an immediate, strict compliance audit with actionable pivots, ensuring your game mechanics are safe and profitable.

## Installation & Usage

You can use this skill with any major LLM (ChatGPT, Claude, Gemini, etc.).

### ChatGPT / Custom GPT
1. Create a new "Custom GPT".
2. Copy the contents of `system_prompt.md` and paste it into the **Instructions** field.
3. Save the GPT and start submitting your design docs.

### Claude (Anthropic)
1. Start a new chat or create a new "Project" in Claude.
2. In the "System Prompt" or "Custom Instructions" section, paste the contents of `system_prompt.md`.
3. Provide your game mechanics for evaluation.

### Gemini (Google)
1. Start a new chat with Gemini Advanced (or use Google AI Studio).
2. Paste the `system_prompt.md` content as your initial prompt to set the persona.
3. Follow up with your mechanic proposals.

### Testing the Skill
You can test the effectiveness of this skill by providing it with the realistic scenario found in `test_case_input.md`.
