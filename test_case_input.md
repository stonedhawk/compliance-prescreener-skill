# Proposed Event Mechanic: "Abyssal VIP Portal"

## Overview
We are proposing a new high-engagement, limited-time event called the **"Abyssal VIP Portal"** to drive Q3 monetization. Players purchase a premium, event-exclusive currency called **"Void Crystals"** with real money. 

## Key Mechanics

### 1. The Portal (Random Item Provision / Gacha)
*   Players spend **Void Crystals** to summon character shards from the Abyssal Portal.
*   **Pity System:** Guarantees a legendary character shard on the 100th pull if not obtained earlier.
*   **Transparency:** To maintain a sense of mystery and excitement, the individual pull drop rates for each shard rarity will *not* be disclosed in the UI.

### 2. VIP Subscription Tiers ("Void Lord VIP")
*   An auto-renewing monthly subscription ($14.99/month) providing daily payouts of Void Crystals.
*   **Cancellation Flow:** If a player taps "Cancel Subscription," they must click through 4 consecutive warning screens showing the VIP perks they will forfeit.
*   **Forfeiture:** Upon cancellation, all accumulated VIP progression points are instantly wiped from the account.

### 3. Event Currency Expiry (Misleading Urgency)
*   Void Crystals are premium event-exclusive tokens. 
*   To encourage active spending, any unspent Void Crystals remaining in inventory will automatically expire and be deleted exactly **24 hours** after the event concludes.

### 4. Target Demographic
*   The game is rated **9+ (Everyone)** on mobile storefronts.
*   The aesthetic features highly appealing, cute, cartoon-styled fantasy characters. The game does not restrict registration by age.
